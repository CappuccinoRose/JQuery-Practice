# jQuery Practice - 交互式前端练习项目

> 一个专注于 jQuery 技术栈的交互式前端练习项目，包含多种常见交互效果的实现。

## 📋 项目概述

本项目是一个综合性的 jQuery 练习集合，旨在通过实际案例帮助开发者掌握前端交互技术的实现方法。每个模块都包含完整的 HTML、CSS 和 JavaScript 代码，可直接运行查看效果。


## ✨ 功能特点

- **3D 旋转木马** - 带倒影效果的 3D 图片轮播
- **电梯导航** - 模拟京东风格的右侧导航
- **全选功能** - 表格复选框全选/反选
- **图片换肤** - 动态切换页面主题
- **背景切换** - 多种背景效果切换
- **导航菜单** - 3D 下拉导航菜单
- **图片轮播** - 手风琴式图片切换
- **倒计时** - 精确时间倒计时
- **表格操作** - 动态增删改查
- **快递查询** - 模拟快递单号查询

## 🚀 使用方法

1. **直接运行**
   - 双击任意 HTML 文件即可在浏览器中查看效果
   - 无需额外安装，所有依赖已包含

2. **修改练习**
   - 打开任意 HTML 文件进行修改
   - 调整 CSS 样式或 JavaScript 逻辑
   - 保存后刷新页面查看效果

## 🔧 技术栈

- **jQuery** - 核心交互库
- **HTML5** - 页面结构
- **CSS3** - 样式与动画
- **JavaScript** - 交互逻辑
